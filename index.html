How To Play

Objective

Fill empty tiles so each city has the correct number of resources. Try to maximize your score by earning bonus points.

Rules

  1. Resources can only be placed on tiles surrounding a city.
  2. Each city must touch exactly its shown number of resource tiles.
  3. Two matching resources cannot touch each other, even diagonally.
  4. If a placed resource touches a port, it must match that port type.

Point System

  • Resource Points: Wheat 20, Sheep 20, Wood 10, Brick 10.
  • Port bonus: +20 if a tile matches any adjacent port.
  • City multiplier: multiply tile points by number of adjacent cities.
